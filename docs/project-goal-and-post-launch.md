# NexusM&A プロジェクトの目標と公開後運用

## プロジェクトの目的

NexusM&Aサイトの目的は、電話営業や紹介依存に偏らず、M&Aや事業承継をまだ具体的に考えていない中小企業経営者との接点をつくること。

特に、いきなり「会社を売りませんか」と迫るのではなく、経営者が自分ごととして考えやすい入口を用意する。

中心導線:

```text
社長不在90日診断
↓
経営課題・承継課題の自己認識
↓
無料相談フォーム
↓
メール中心の初回接点
↓
M&A・事業承継・第三者承継・社内承継などの選択肢整理
```

## 事業上の目標

- 譲渡候補先、譲受候補先、通常問い合わせの自然流入を増やす
- 営業電話に頼らず、問い合わせフォームと匿名診断から相談を生む
- `sasa-eru.com` をNexusM&Aの本命ドメインとして育てる
- 中小M&Aガイドライン遵守、運営者情報、料金説明を明示し、信頼性を担保する
- すぐにM&Aを考えていない経営者にも「まず診断してみる」行動を促す

## このサイトで重視するユーザー

### 売り手・譲渡候補

- 後継者不在だが、M&Aまではまだ考えきれていない経営者
- 従業員、取引先、社名、経営者保証を心配している経営者
- 電話営業に抵抗があり、まずはメールや匿名診断から始めたい人

### 買い手・譲受候補

- 成長戦略として中小M&Aを検討している企業
- 希望業種、地域、規模を登録したい企業
- まだ具体案件ではなく、方針整理から入りたい企業

### その他

- 提携先、士業、金融機関、メディア、採用・協業の問い合わせ

## 次にやるべき改善

### 1. Search Console初期確認

サイトマップ送信後、1〜3日後に確認する。

- サイトマップが成功になっているか
- `https://sasa-eru.com/` がインデックス登録対象になっているか
- クロールエラーが出ていないか

### 2. 問い合わせフォームの本番再確認

独自ドメイン化後に再度確認する。

- 通常問い合わせ
- 売り手相談
- 買い手相談
- 診断結果からの問い合わせ引き継ぎ

### 3. コラム追加

検索流入を狙う記事を追加する。

優先テーマ:

- 後継者不在 会社 どうする
- 事業承継 M&A メリット デメリット
- 中小企業 M&A 相談前に準備すること
- 経営者保証 M&A
- 社長しか分からない会社の承継リスク
- M&A 電話営業が嫌な経営者向けの相談方法

### 4. 通常URL化

コラム記事50本は通常URL化済み。

例:

```text
https://sasa-eru.com/columns/succession-local-successor-shortage/
```

会社概要などの固定ページは引き続きハッシュルーティングのため、必要性とSEO効果を見ながら通常URL化を判断する。

## 次回以降のコンテンツ制作計画

### 基本方針

- 「M&A・事業承継」という言葉だけでは動かない経営者へ、具体的な環境変化や困り事から接点をつくる
- 社長は必ずしも引退を望んでいるとは限らないため、肩書・収入・居場所・役割を残す選択肢も示す
- 右腕育成を解決策の中心にせず、社長の機能を吸収・分担できる買い手とのマッチングを重視する
- 売り手に負担をかける分析項目は、表側では本人に意味のある問いへ変換する
- 恐怖を過度に煽らず、事実・選択肢・注意事項を示して相談へつなげる

### 第1段階: 案件に近い環境変化別ページ

次回はここから着手する。通常コラムではなく、相談へつながる専用ページまたは重点記事として作成する。

1. 社長を続けながら会社を譲渡できるのか
2. 後継者候補が辞退したときの選択肢
3. 社長が病気・入院した会社の事業承継
4. 資金繰りが厳しい会社でもM&Aできるのか
5. 右腕・幹部・熟練社員が退職した会社の対応
6. 廃業を決める前に確認すべき第三者承継
7. 経営者保証を家族へ残さないための準備

### 第2段階: 社長依存企業への回答

8. 顧客が社長についている会社の引き継ぎ方
9. 社長依存企業に合う買い手の条件
10. 即時退任型と段階退任型M&Aの違い
11. 社長の機能を買い手組織へ分散する方法
12. 営業・技術・判断・金融機関対応・地域信用の依存分析

### 第3段階: 買い手側コンテンツ

13. 重要な取引先をM&Aで守る方法
14. 後継者不在の取引先を承継するメリット
15. 採用よりM&Aが有効になるケース
16. 売り手社長へ一定期間残ってもらう方法
17. 売り手の社長依存を吸収できる買い手とは

### 第4段階: 信頼を補強する無料資料

18. 社長不在90日チェックシート
19. PDF小冊子「経営者が90日間、業務を離れることになったら」
20. M&A後の社長の残り方比較表
21. 家族向けの保証・借入・株式確認表
22. 緊急時の銀行・給与・取引先対応チェック表

PDF小冊子は営業先の社員へ直接配布しない。サイト上の無料資料として掲載し、経営者本人、家族、後継候補、専門家が必要に応じて利用できる位置づけにする。M&A案件獲得の主力ではなく、信頼形成と90日診断の補助資料として扱う。

### 第5段階: 診断・マッチング機能

23. 社長の残り方診断
24. 社長依存度診断
25. 買い手企業の承継吸収力診断
26. 売り手の依存要素と買い手の組織力のマッチング
27. 匿名の将来承継意向登録
28. 買い手の承継希望条件登録

### 次回の開始地点

第1段階の7テーマについて、以下を決めてから順次制作する。

1. 専用ランディングページにするか通常コラムにするか
2. 検索意図と想定読者
3. 法務・税務上の断定を避ける注意事項
4. 記事ごとの問い合わせCTA
5. 関連記事・診断・問い合わせフォームへの内部導線

## 判断基準

このサイトの改善では、次の基準を優先する。

- 経営者が警戒せずに読み進められるか
- 電話営業っぽさを消せているか
- 相談前の不安が減るか
- ガイドライン遵守・運営者情報・料金が明確か
- 診断から問い合わせへの導線が自然か
- 実在実績に見える誤解や誇大表現がないか
